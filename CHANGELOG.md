## [0.1.0] — 2026-05-14

### Добавлено

- Раздел **«Память»**: оценка занятого места по типовым категориям тома, круговая диаграмма и карточки разделов; занятое, свободное и общее место на томе; сохранение снимка оценки и сводки быстрой очистки (UserDefaults).
- Раздел **«Быстрая очистка»**: план с оценками и исключаемыми путями, перенос в корзину и отдельный сценарий безвозвратной очистки корзины с подтверждением; эвристики «можно / не трогать»; подсказки по крупным файлам и папкам.
- Режим **обзора с корня `/`**, навигация по дереву внутри выбранного корня (`FolderPathScope`), поверхностный листинг с размерами.
- **`scripts/package-macos-app.sh`** и **`.github/workflows/release.yml`**: сборка Release, ZIP с `MacosStorageCleaner.app`, публикация архивов при push тега `v*`.
- Экраны **«Настройки»** и **«О программе»**; README с инструкцией установки и ссылкой на стабильный архив `MacosStorageCleaner-macos.zip`.

### Ранее (вошло в первый релиз)

- Выбор папки через `NSOpenPanel`, таблица первого уровня, `ShallowDirectoryListingService`.
- Модульный Swift Package (`CleanerCore`, `CleanerUI`, `CleanerApp`), спецификация и документация для участников.
